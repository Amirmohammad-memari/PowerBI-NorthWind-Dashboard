# PowerBI-NorthWind-Dashboard
Interactive Power BI dashboard built using the Northwind database.
This is my second dashboard developed using Microsoft Power BI, built using the Northwind sample database.
Northwind is a well-known business database that contains sales, customers, products, orders, employees, and shipping data of a fictional company, making it a great dataset for business intelligence and data analysis projects.

In this dashboard, I utilized a variety of built-in and custom visuals to create engaging and interactive data visualizations.

Additionally, I used DAX to build dynamic visuals, create new tables, columns, and measures, and enhance the analytical capabilities of the dashboard.

To improve the user experience and provide deeper insights, custom tooltips were also designed for several charts and visual elements.
